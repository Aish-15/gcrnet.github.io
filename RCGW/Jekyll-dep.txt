About Ruby, Gems, Bundler, and other prerequisites

Ruby is a programming language you must have on your computer in order to build Jekyll locally.
Ruby has various gems (or plugins) that provide various functionality. Each Jekyll project usually
requires certain gems. More information from the link below:

https://idratherbewriting.com/documentation-theme-jekyll/mydoc_about_ruby_gems_etc.html

To publish the site locally on your computer, run the following command

bundle exec jekyll serve
 the following is returned

            Source: /Users/j_fossot/RCGW
       Destination: /Users/j_fossot/RCGW/_site
 Incremental build: disabled. Enable with --incremental

    Server address: http://127.0.0.1:4000/

copy and paste the server address as displayed in the address
bar of your browser to publish the site
